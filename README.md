# Gestão Clínica Inteligente - Sempre Vida 🏥

Sistema desenvolvido para automação de processos clínicos e gestão de agendamentos utilizando **Django** e **IA Agêntica**.

## 🛠️ Tecnologias
- **Backend:** Django (Python)
- **Banco de Dados:** Supabase (PostgreSQL)
- **Inteligência Artificial:** Agentes autônomos para triagem e alertas médicos (n8n + OpenAI)

## 🎯 Objetivos do Projeto
- Automatizar o fluxo de agendamentos.
- Monitorar resultados de exames (ex: Sangue Oculto) via IA.
- Gerar relatórios gerenciais em tempo real.
